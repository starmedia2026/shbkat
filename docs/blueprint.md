# **App Name**: Shabakat

## Core Features:

- User Authentication: Secure user registration and login functionality.

## Style Guidelines:

- Primary color: Deep blue (#3B5998) to convey trust and security, inspired by the essence of networking and connection.
- Background color: Light gray (#F2F2F2) to provide a clean, professional backdrop.
- Accent color: Soft green (#9ACD32) to highlight interactive elements and call-to-action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif font with a neutral and modern appearance for a clean user experience.
- Clean and intuitive layout for easy navigation. Emphasize clear calls to action.
- Use modern, outline-style icons to represent various actions and categories within the application.
